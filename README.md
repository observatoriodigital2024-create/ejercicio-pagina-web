# Despliegue en GitHub Pages
Este proyecto ya está listo para publicarse con GitHub Pages usando la carpeta `docs/` como raíz del sitio.
> **Pasos para habilitar**
1. En GitHub, abre el repositorio `observatoriodigital2024-create/ejercicio-pag-wb`.
2. Ve a Settings > Pages.
3. En Source, selecciona la rama `main` (o `master`) y la carpeta `/docs`.
4. Guarda los cambios.
5. Espera unos minutos y accede a `https://observatoriodigital2024-create.github.io/ejercicio-pag-wb/`.
